# LSTM_Speed_Predict
## 1. 简介
本项目是一个基于LSTM的叉车臂速度预测模型，主要用于实现油泵电机的速度跟随和负载重量分类预测。

## 2. 数据集
本项目使用的数据集是一个包含多个变量的时间序列数据集，其中包括油泵电机的实际速度、控制速度、期望速度与目标高度差。数据位于data文件夹中，train和test。
